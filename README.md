Id3 Tag Music Renamer
====================

Application that renames the music (ID3Tag) from the file name. 

Example:

	Filename: Avicii - Wake Me Up.mp3
	
	Artist: Avicii
	
	Music: Wake Me Up


Third Party Libraries	
====================	
* commons-httpclient-2.0.2
* JID3
* json-20090211
